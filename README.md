🚦 AI Junction Optimization System

Transforming city traffic into an intelligent, self-optimizing network.

The AI Junction Optimization System is a smart-city platform that reimagines how road intersections operate. Instead of relying on fixed signal timers, it analyzes live traffic conditions, predicts congestion before it happens, and dynamically adjusts signals to keep vehicles moving efficiently.

🧠 What Makes It Unique?

At the core of the system is the Traffic Pressure Index (TPI) — a custom AI scoring model that combines vehicle count, waiting time, predicted traffic, and emergency priority to decide which lane should receive the green signal first.

🚑 Built for Real-World Impact

Creates Emergency Corridors for ambulances and fire trucks.
Recommends the fastest, least-congested route using live location.
Displays city-wide traffic heatmaps and predictive analytics.
Explains every AI decision with transparent reasoning.
Estimates CO₂ savings from reduced idle time.
